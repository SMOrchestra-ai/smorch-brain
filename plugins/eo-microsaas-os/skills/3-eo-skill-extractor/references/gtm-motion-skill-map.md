**Used by:** 3-eo-skill-extractor when recommending which skills students should build first.

## How To Use This Table

1. Read the student's selected motion from `2-GTM/output/GTM/`
2. Find the matching row above
3. Present the 3 suggested skills as options
4. Let the student pick one to build first (or propose their own)
5. Save completed skills to the Category column's subfolder in `3-Newskills/`

## Universal Skills (Suggest Regardless of Motion)

These are useful for any founder:
- **CRM contact updater** — keeps lead data fresh across tools (Ops)
- **Weekly metrics compiler** — pulls numbers from multiple tools into one report (Ops)
- **Invoice/proposal generator** — creates branded docs from templates (Ops)
