# GTM Fitness Output Template: gtm-fitness.md

This document defines the structure and content format of the output file that founders receive after completing the GTM Fitness assessment.

---

## Output File: gtm-fitness.md

**File Location:** Stored in founder's project directory or provided as download
**File Naming:** `gtm-fitness-[founder-name]-[date].md`
**Audience:** Founder + advisors
**Format:** Markdown with clear sections, tables, and action items

## Output Template: Key Formatting Rules

1. **Use emojis for section clarity** (✅, ⚠️, 📊, 🚀, etc.)
2. **Include tables for easy scanning** (rankings, tools, timelines)
3. **Bold critical information** (scores, tier assignments, 72-hour commitment)
4. **Use [brackets] for variable insertion** by Claude during assessment
5. **Keep language founder-friendly** (not jargon-heavy)
6. **Include "Why This Works For You"** section for each motion (explains connection to their specific assessment answers)
7. **Provide specific, actionable 72-hour items** (not vague guidance)
8. **Reference the references/ folder** at end so founder knows where to find deep content

---

## Output File Variations

### Variation 1: Quick Summary (1-page)
**For busy founders:** Top 3 motions, 72-hour commitment, timeline

### Variation 2: Deep Dive (5-page)
**For founder + advisor:** Full scoring, all 13 motions, all gates, full context

### Variation 3: HTML Dashboard
**For visual learners:** Interactive ranking table, motion comparison, timeline visualization

**Default:** Provide Deep Dive (full markdown file). Offer Quick Summary as supplementary.
