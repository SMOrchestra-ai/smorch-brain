**Best for:** Products where buyers actively search for solutions. Bottom-of-funnel content captures ready-to-buy traffic.

### 2. BOFU Content Templates (DOCX)
**File:** `bofu-seo/bofu-content-templates.docx`

3 article templates optimized for conversion.

```
TEMPLATE 1: COMPARISON PAGE
"[Your Product] vs [Competitor]: Which is right for [ICP]?"
- H1: Comparison title
- Intro: Why someone would compare these two
- Feature table: side-by-side comparison
- Pricing comparison
- Who should use [Competitor]: honest assessment
- Who should use [Your Product]: your strengths
- Verdict: when to pick each
- CTA: Try [Your Product] free

TEMPLATE 2: USE CASE PAGE
"How [ICP] Uses [Your Product] to [Outcome]"
- H1: Outcome-focused title
- The problem (from ICP pains)
- The old way vs the new way
- Step-by-step walkthrough
- Results and metrics
- CTA: Start for free

TEMPLATE 3: ALTERNATIVES ROUNDUP
"Top [X] [Product Category] Tools for [ICP] in [Year]"
- H1: Roundup title
- Your product listed #1 (with honest positioning)
- 4-6 competitors with fair descriptions
- Comparison criteria: features, pricing, pros/cons
- "Best for" recommendation per tool
- CTA: Why we recommend starting with [Your Product]
```

### 3. Technical SEO Checklist (DOCX)
**File:** `bofu-seo/seo-technical-checklist.docx`

```
PAGE-LEVEL SEO CHECKLIST:
□ Title tag: keyword + brand (under 60 chars)
□ Meta description: benefit + CTA (under 155 chars)
□ H1: one per page, includes primary keyword
□ URL: short, keyword-included, no dates
□ Internal links: 3-5 to related pages
□ Schema markup: Article, Product, or FAQ
□ Page speed: under 3 seconds
□ Mobile responsive
□ Image alt tags: descriptive, keyword-relevant

SITE-LEVEL SEO:
□ Sitemap.xml submitted to Google Search Console
□ Robots.txt properly configured
□ SSL certificate active
□ Core Web Vitals passing
□ Arabic content: hreflang tags if bilingual
□ Local SEO: Google Business Profile if applicable
```

## MENA Adaptations

- Arabic SEO is a blue ocean — most MENA SaaS founders only target English keywords
- "best [X] for Saudi Arabia" or "best [X] for UAE" keywords have almost zero competition
- Arabic content should use MSA for SEO (searchable) but conversational tone in the writing
- Google is dominant in MENA (90%+ search share) — focus efforts there
