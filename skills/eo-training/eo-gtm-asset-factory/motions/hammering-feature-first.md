# Hammering-Feature-First Launches — Motion Assets

**Best for:** Products with clear feature differentiation that can be demonstrated quickly.
**MENA Note:** Feature launches work when combined with trust signals. Don't just ship — show proof.

---

## Assets Produced

### 1. Feature Launch Playbook (DOCX)
**File:** `hammering-feature-first/feature-launch-playbook.docx`

```
THE FEATURE LAUNCH FRAMEWORK:

PHASE 1: BUILD ANTICIPATION (1 week before)
- LinkedIn teaser: "Working on something for [ICP] who struggle with [pain]..."
- Email preview to waitlist: "Coming next week: [feature outcome]"
- Community hint: Share a screenshot or GIF of the feature in progress

PHASE 2: LAUNCH DAY
- LinkedIn post: Outcome demo (before/after with new feature)
- Email blast: "New: [feature] — [outcome] in [timeframe]"
- Product Hunt / relevant directory listing
- WhatsApp to warm contacts: quick demo video
- Community: detailed walkthrough post

PHASE 3: AMPLIFY (1 week after)
- User testimonial collection (reach out to beta testers)
- LinkedIn post: "X people already using [feature] — here's what they're saying"
- YouTube tutorial: detailed how-to using the new feature
- Twitter thread: build-in-public story of why you built it

PHASE 4: ITERATE (2 weeks after)
- Collect feedback systematically
- Share roadmap update based on feedback
- Tease next feature based on user requests
```

### 2. Feature Announcement Templates (DOCX)
**File:** `hammering-feature-first/feature-announcement-templates.docx`

```
EMAIL ANNOUNCEMENT:
Subject: "New: [Feature name] — [specific outcome]"
Body: What it does (1 sentence) + Why it matters for [ICP] (1 sentence) +
How to use it (3 steps) + CTA (try it now)

LINKEDIN POST:
"Shipped something new today.
[ICP] told us [pain]. So we built [feature].
Now you can [outcome] in [timeframe].
[Demo link or screenshot]
Try it: [link]"

TWITTER THREAD:
"🚀 Just shipped: [feature name]
Here's what it does and why [ICP] needs it 🧵
[5-7 tweet thread walking through the feature + outcomes]"

WHATSAPP:
"Hey [Name] — just launched [feature].
Does [outcome] for you automatically.
[Screenshot/GIF]
Try it: [link]"
```

### 3. Launch Week Content Calendar (DOCX)
**File:** `hammering-feature-first/launch-week-calendar.docx`

Day-by-day content plan for a feature launch week.

| Day | Channel | Content | Goal |
|-----|---------|---------|------|
| Mon | LinkedIn + Email | Teaser: "Something's coming" | Build anticipation |
| Tue | All channels | Launch announcement + demo | Drive signups |
| Wed | YouTube | Detailed tutorial video | SEO + education |
| Thu | LinkedIn | User reaction / testimonial | Social proof |
| Fri | Community + WhatsApp | Q&A + personal outreach | Convert |
| Sat | Twitter thread | Build story behind the feature | Engagement |
| Sun | Review metrics + plan next feature | Internal | Learn |

## MENA Adaptations

- Feature launches in Arabic communities get much more organic sharing
- WhatsApp screen recordings of new features perform better than text announcements
- Product Hunt launches should be paired with Arabic community amplification
