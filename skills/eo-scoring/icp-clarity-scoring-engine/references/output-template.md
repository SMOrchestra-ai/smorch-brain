## Formatting Guidelines

**For score bands in template:**
- **Elite (85-100):** Use all recommendation sections; confidence is high
- **Strong (70-84):** Flag 1-2 sections for refinement; proceed to SC3
- **Adequate (55-69):** Flag 2-3 sections requiring rework; 1 revision round recommended
- **Weak (40-54):** Major gaps; recommend customer discovery before resubmitting
- **Foundational (0-39):** Start over; not yet ready for scoring

**Consistency flags to include:**
- Any red/yellow flags from consistency engine (Sections A-E cross-validation)
- Founder's response to flags (e.g., "Acknowledged — will update positioning")
- Rationale if founder disagrees with flag (e.g., "Actually, our budget target IS <$50/mo because...")

**Recommendation specificity:**
- Tie each recommendation to a specific question (e.g., "A2 scores 2/5 because...")
- Suggest action (e.g., "Next step: conduct 3 day-in-the-life interviews")
- Connect to impact (e.g., "This will improve messaging clarity for GTM")
