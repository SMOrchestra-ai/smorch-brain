## Calibration Notes

These benchmarks are v1. They should be updated as real execution data accumulates.

**How to update:**
1. After each completed ticket, compare actual time vs estimate
2. If actual differs by > 30%, log the variance in a comment
3. Monthly: Claude reviews all variance data and proposes benchmark adjustments
4. Mamoun approves updated benchmarks

**Factors that affect estimates:**
- First time doing a task type: add 50% buffer
- Arabic content: add 15-20% over English equivalent
- Client-specific customization: add 30% over template-based work
- Cross-tool integration: add 25% for handoff complexity
- New tool (first 3 uses): add 50% for learning curve
